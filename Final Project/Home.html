Black Ops 7

Black Ops 7 changed the game forever but it still was our beloved call of duty. From new guns, as usual, to a whole new movement system call omnimovement which was a dissapointment to some fans but others loved it. But, due to the whole new movement system the game did loose a lot of long time gamers. That will all change with thier new installment, Call of Duty: Modern Warfare 4 which I can not wait for and a lot of fans feel the same way.
